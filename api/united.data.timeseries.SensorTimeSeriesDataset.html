SensorTimeSeriesDataset#

class united.data.timeseries.SensorTimeSeriesDataset(data: DataFrame, group_ids: List, time_idx: str, min_encoder_length: int, max_encoder_length: int, min_decoder_length: int, max_decoder_length: int, sensors: List[str], encoder_variables: List[str], decoder_variables: List[str], targets: List[str], static_variables: List[str] = [], training=True, random_length=True, scalers=None, target_normalizers=None, imputers: str | Dict[str, SimpleImputer | BaseTransformer] | None = None, add_na=False, ignore_na=False, allow_missing_time_steps=False, unknown_value=nan, non_overlapped_windows=False, return_missing_mask=True)#

Bases: TimeSeriesDataset

Pytorch Dataset for timeseries data TimeSeriesDataset.to_dataloader() method to convert dataset into a dataloader

Parameters:

  • data (pd.DataFrame) – DataFrame with time series data. Combine with time_idx and group_ids to determine sequences

  • group_ids (List) – list of column names are used to identify a time series

  • time_idx (str) – column name of integer column that is used as time dimension of sequence.

  • min_encoder_length (int) – minimum encoder length

  • max_encoder_length (int) – maximum encoder length

  • min_decoder_length (int) – minimum decoder length

  • max_decoder_length (int) – maximum decoder length

  • sensors (List[str]) – list of column names are used as sensor

  • encoder_variables (List[str]) – list of column names are used for encoding

  • decoder_variables (List[str]) – list of column names are used for decoding

  • targets (List[str]) – list of column names are used as prediction targets

  • static_variables (List[str], optional) – static variables which are constant at all time steps. Defaults to [].

  • training (bool, optional) – flag if dataset is used for training or not. Defaults to True.

  • scalers (_type_, optional) – dictionary of scikit-learn compatible scalers of encoder/decoder variables. Defaults to None.

  • target_normalizers (_type_, optional) – dictionary of scikit-learn compatible scalers of targets. Defaults to None.

  • imputers (Union[ str, Dict[str, Union[SimpleImputer, BaseTransformer]] ], optional) – dictionary of scikit-learn compatible imputers. Defaults to None.

  • add_na (bool, optional) – if to force encoding of nan at 0 for categorical variables only . Defaults to False.

  • ignore_na (bool, optional) – if to ignore na values and map them to zeros (this is different to add_nan=True option which maps ONLY NAs to zeros while this options maps the first class and NAs to zeros). Defaults to False.

  • allow_missing_time_steps (bool, optional) – if True, accept missing data and imputers will be used to fill missing data. Defaults to False.

  • unknown_value (_type_, optional) – value which will be used for filling in in case of unknown categorical values. Defaults to np.nan.

  • non_overlapped_windows (bool, optional) – sample non-overlapped samples when do sliding window. Defaults to False.

  • return_missing_mask (bool, optional) – flag to return missing mask of data, some models such as GraphIC is required this mask. Defaults to False.
